FAQ

help_outline Questions frequentes

Comment puis-je voir les utilisateurs qui seront supprimes ? expand_more

Pour des raisons de confidentialite, les adresses email ne sont pas affichees dans ce tableau de bord. Vous pouvez consulter la liste des utilisateurs directement dans BigQuery en executant la requete suivante :

-- Executez dans BigQuery (projet: oa-data-coepowerbi-np)
-- Retourne les emails des utilisateurs inactifs eligibles a la suppression
SELECT
  TRIM(LOWER(user_email)) AS user_email,
  last_workspace_pro_consumer_activity,
  last_workspace_pro_builder_activity,
  DATE_DIFF(CURRENT_DATE(), start_date, DAY) AS nb_jours_licence
FROM
  `itg-btdppublished-gbl-ww-pd.btdp_ds_c1_0a2_powerbimetadata_eu_pd.license_pro_users_v1`
WHERE
  end_date = '9999-12-31'
  AND DATE_DIFF(CURRENT_DATE(), DATE(last_workspace_pro_consumer_activity), DAY) > 120
  AND DATE_DIFF(CURRENT_DATE(), DATE(last_workspace_pro_builder_activity), DAY) > 120
  AND pbi_asset_ownership_details IS NULL
  AND DATE_DIFF(CURRENT_DATE(), start_date, DAY) > 60
  AND STRPOS(REPLACE(TRIM(LOWER(user_email)), '@loreal.com', ''), '.') > 0
ORDER BY nb_jours_licence DESC;

Explication : Cette requete retourne les utilisateurs qui ont une licence Pro depuis plus de 60 jours, sans activite Consumer ni Builder depuis 120 jours, et qui ne possedent aucun asset Pro (workspace, rapport, modele semantique). Seuls les comptes humains (avec un '.' dans l'email) sont inclus.

Comment puis-je voir les utilisateurs qui ont ete supprimes ? expand_more

L'historique des executions ci-dessus montre les comptes agreges (nombre d'utilisateurs identifies et supprimes par execution). Pour retrouver la liste detaillee des emails supprimes :

1

Fichier CSV local

Chaque execution (dry-run ou live) exporte un fichier CSV dans le dossier ./output/. Ce fichier contient les emails, le statut et la date d'execution.

2

Cloud Logging (apres deploiement GCP)

Une fois deploye sur Cloud Run, l'historique complet sera disponible dans Cloud Logging avec les details de chaque suppression.

A quelle frequence le nettoyage s'execute-t-il ? expand_more

Par defaut, le nettoyage automatique s'execute le dernier jour de chaque mois a 02h00 UTC. Vous pouvez modifier cette frequence dans la section Configuration > Programmation (premier jour du mois, chaque lundi, etc.).

En complement, le bouton "Executer la suppression" permet de lancer le nettoyage manuellement a tout moment.

warning

Confirmer la suppression

Vous etes sur le point de supprimer les utilisateurs inactifs du groupe de licence Pro.

Cette action est irreversible. Les utilisateurs devront etre re-ajoutes manuellement si necessaire.
