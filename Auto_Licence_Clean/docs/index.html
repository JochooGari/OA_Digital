API Catalogue
GET /groups/{group_email}/members
check_circle

List all members of a group. Supports ?source=gcp|azure|both_common filter.

curl -X GET \
  "https://api.loreal.net/global/it4it/itg-groupsapi/v1/groups/MY-GROUP@loreal.com/members" \
  -H "Authorization: Bearer $(gcloud auth print-access-token)"
DELETE /groups/{group_email}/members
check_circle

Remove members from a group. Service account must be owner of the group.

curl -X DELETE \
  "https://api.loreal.net/global/it4it/itg-groupsapi/v1/groups/MY-GROUP@loreal.com/members" \
  -H "Authorization: Bearer $(gcloud auth print-access-token)" \
  --json '{"members": ["john.doe@loreal.com"]}'
POST /groups/{group_email}/members
check_circle

Add members to a group.

curl -X POST \
  "https://api.loreal.net/global/it4it/itg-groupsapi/v1/groups/MY-GROUP@loreal.com/members" \
  -H "Authorization: Bearer $(gcloud auth print-access-token)" \
  --json '{"members": ["john.doe@loreal.com"]}'
GET /groups/{group_email}/authorized
check_circle

List group owners (principals authorized to add/remove members).

curl -X GET \
  "https://api.loreal.net/global/it4it/itg-groupsapi/v1/groups/MY-GROUP@loreal.com/authorized" \
  -H "Authorization: Bearer $(gcloud auth print-access-token)"
